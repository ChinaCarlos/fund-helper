# 功能说明

## Web 应用

### 持仓监控（`/`）

- 账号密码登录（管理员可管理用户）
- 养基宝微信扫码绑定
- 多账户 Tab、汇总卡片、基金表格列自定义
- 分组收益曲线（SVG 自研图表）

### 市场数据

- **市场排行**（`/market`）：全市场基金多维度排行
- **板块热力图**（`/market/heatmap`）：行业/概念板块涨幅或资金流向

### 通知推送

- 钉钉 / 飞书 / 企业微信
- 手动刷新后 / 定时 / 仅交易时段

## 浏览器插件

- 微信扫码登录，Token 存 `chrome.storage.local`
- 四大指数、总资产、当日收益、基金排序
- 净值已更新显示「已更新」，交易中显示「预估」
- 详见 [Chrome 插件文档](/clients/chrome-extension)

## VS Code / Cursor 扩展

- 活动栏、状态栏、底部 Panel、编辑器标题栏四个入口
- 状态栏悬浮展示持仓摘要（Markdown + HTML 表格）
- 启动时自动激活，主题色适配编辑器
- 详见 [VS Code 扩展文档](/clients/vscode-extension)

## 桌面端

- Tauri + Rust，本地 SQLite 存登录态与通知配置
- 收益曲线、系统托盘、飞书应用 IM 卡片
- 详见 [桌面端文档](/clients/desktop)
